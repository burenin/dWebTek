The Wiki Language

Wiki articles are normally authored in a non-XML syntax, where:

  • An image looks like [[Image:url]].
  • Links look like [[word]], [[wiki:word]], and [url word].
  • The italics style is selected as ''...'', and the bold weight as '''...'''.
  • The fixed-width style is selected as <tt>...</tt>.
  • A header is written as == ... == at the beginning of a line.
  • List of items is written as:
    * first
    * second
    * third
    where the *'s must begin the lines.
  • A horizontal rule is denoted by ---- which must begin the line.
  • A character reference is denoted as e.g. &oslash;.
  • A linebreak is written as <br/>.
  • Texts and whitespace are written literally.

Example of an article written in the wiki syntax:

== The dWebTek Project ==
The [http://www.cs.au.dk/dWebTek/project.html project] involves:
* a Wiki [[server]]
* a Wiki [[metaserver]]
* a Wiki [[client]]
Pages are written in a subset of ''Wikipedia'' syntax and are translated into WikiXML using a Java class <tt>Txt2WikiXML</tt>.
----
Note that the parser does '''not''' generate any syntax errors.
<br/>
Once the system is operational, you can write lots of Wiki pages.

Creating, Editing & Reading Articles

Use "_" to denote a space when trying to reach or create an article. E.g. an article on "computer science" is denoted "computer_science".

Article names can only contain letters (a-zA-Z) and the underscore character (_).
